# Semiconductor Conductivity Simulator

A dark, high-impact scientific dashboard for intrinsic semiconductor temperature dependence, built with React, Tailwind CSS, Vite, Framer Motion, Recharts, and Lucide React icons.

## Run locally

1. Install Node.js and npm.
2. Open a terminal in `websitephysics`.
3. Run:
   ```bash
   npm install
   npm run dev
   ```
4. Open the localhost address shown by Vite.

## Features

- Temperature slider between 250 K and 1000 K
- Animated semiconductor band structure visualization
- Live conductivity, bandgap, carrier concentration, and mobility graphs
- Material comparison mode for Si, Ge, GaAs, GaN, SiC
- Intrinsic / n-type / p-type doping simulation
- CSV export and print-ready report
- Educational physics commentary and AI-style insights
